# Hello World React App

A simple React app displaying "Hello World" for learning purposes.

## 🚀 Features

- Renders "Hello World" in an `<h1>` tag
- Built with React 18+
- Minimalist design

## 📦 Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/AbbeyCoder/hello-world-react.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm start
   ```

## 🛠️ Technologies Used

- React
- Create React App
- JavaScript/JSX
